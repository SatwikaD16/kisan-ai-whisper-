# Kisan AI Whisper

This is the Whisper speech-to-text module we are using for Kisan AI.

It takes an audio file as input and converts the speech into text.

Requirements:

* Python 3.10+
* FFmpeg
* Whisper

Install dependencies:

pip install -r requirements.txt

Run:

python speech_to_text.py

Current status:

* Whisper installed and tested
* Audio to text conversion working successfully

Example:

Audio: "What is the weather today?"

Output:
"What is the weather today?"
